Ctrl+Alt+T

Remove Older Tor
$ cd ~ && rm -r .tor-browser-en[ENTER]

Install New
$ sudo add-apt-repository ppa:webupd8team/tor-browser[ENTER]
$ sudo apt-get update[ENTER]
$ sudo apt-get install tor-browser[ENTER]
